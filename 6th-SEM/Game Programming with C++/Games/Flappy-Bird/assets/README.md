# Flappy Bird – Asset Folder

Place the following files in this directory before compiling:

| File               | Dimensions | Notes                          |
|--------------------|------------|-------------------------------|
| `bird.png`         | 141 × 180 px | Bird sprite                  |
| `pipe.png`         | 321 × 52 px  | Pipe segment sprite          |

If either image is missing the game will still run using coloured
rectangles as fallback shapes (yellow bird, green pipes).

## Sounds (optional – placeholder)

Place audio files here if you wish to extend the game with sound effects:
- `flap.wav`   – flap sound
- `hit.wav`    – collision sound
- `score.wav`  – passing a pipe
- `bg_music.ogg` – background music loop
